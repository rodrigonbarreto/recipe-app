# Getting Started with Front-end Recipes project

 [API Repository](https://github.com/rodrigonbarreto/recipes_api).

This project requires these versions:
- node version: v18.12.0
- npm version: 7.21.1

In the project directory, you can run:
```shell
npm install
npm start
```
